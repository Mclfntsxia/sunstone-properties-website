# Sunstone Properties

Real Estate Website Design & Development

A modern real estate website designed in Framer that showcases premium residential properties through a clean, luxurious, and responsive online experience.

---

## Project Overview

Sunstone Properties is a concept real estate website created to help prospective homebuyers browse featured listings, explore services, and connect with experienced real estate professionals.

The project focuses on creating a trustworthy and visually appealing online presence that encourages property inquiries and consultations.

---

## My Role

As the Website Designer & Developer, I was responsible for:

- Planning the website structure and user journey
- Designing the homepage
- Building the website using Framer
- Creating property listing layouts
- Organizing content for readability
- Designing responsive layouts
- Improving UI/UX
- Applying a luxury visual identity
